**Descrição:**
O **Gerador de Charadas** é uma aplicação web interativa que apresenta charadas aleatórias para o usuário. O sistema consome uma API que fornece perguntas e respostas, exibindo-as em um card interativo que gira ao ser clicado para revelar a resposta.

O objetivo do projeto é demonstrar conceitos fundamentais do desenvolvimento web, como consumo de API, manipulação do DOM e criação de interfaces interativas para o usuário.

## ⚙️ Funcionalidades

* 🎲 Gerar charadas aleatórias através de uma API.
* 🎴 Card interativo que gira para revelar a resposta.
* 🔄 Botão **Nova Charada** para carregar uma nova pergunta.
* 🙄 Botão **Não gostei dessa** para pular rapidamente para outra charada.
* 📱 Interface simples e responsiva, sem contar no contador como charada já vista.
