## 📖 Descrição

O **Gerador de Charadas** é uma aplicação web interativa que apresenta charadas aleatórias para o usuário.
O sistema consome uma API que fornece perguntas e respostas, exibindo-as em um card interativo que gira ao ser clicado para revelar a resposta.

O objetivo do projeto é demonstrar conceitos fundamentais do desenvolvimento web, como:

* Consumo de API
* Manipulação do DOM
* Criação de interfaces interativas

## ⚙️ Funcionalidades

* 🎲 Gerar charadas aleatórias através de uma API.
* 🎴 Card interativo que gira para revelar a resposta.
* 🔄 Botão **Nova Charada** para carregar uma nova pergunta.
* 🙄 Botão **Não gostei dessa** para pular rapidamente para outra charada sem contar no contador.
* 📱 Interface simples e responsiva.

## 🔗 API Utilizada

Este projeto utiliza uma API própria desenvolvida para fornecer as charadas.

**Base da API:**
https://api-charadas-two.vercel.app

### Endpoints principais

* **GET /charadas**
  Retorna todas as charadas cadastradas.

* **GET /charadas/aleatoria**
  Retorna uma charada aleatória.
